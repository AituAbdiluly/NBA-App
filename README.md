# NBA-App
### NBA app with ReactJS.

### Installation

create-react-app is required for this project.

```sh
$ sudo npm i -g create-react-app
```

Install the dependencies.

```sh
$ npm install
```

#### Run the project:

```sh
$ npm start
```
And same time it needs to run json-server:

```sh
$ npm run dev-serv
```
